# Stellar Agent Frontend

Minimal **Next.js** frontend for the Stellar Agent project. The codebase has been migrated to **TypeScript**, with `.tsx` pages and components. This repository contains a basic scaffold that can be extended with wallet connectivity, transaction simulations, and other blockchain-related UI features.

## Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm (v9+)

### Setup

```bash
# install dependencies from repo root
npm install

# start development server from app folder
cd apps/stellarAgent-frontend
npm run dev
```

The app will be available at `http://localhost:3000` when running from the app directory.

### Available Scripts

* `npm run dev` – start development mode with hot reloading
* `npm run build` – compile production build
* `npm run start` – run production build locally
* `npm run lint` – run Next.js/ESLint checks
