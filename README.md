# agent-web3-ui

Minimal **Next.js** frontend for the Stellar Agent project. This repository contains a basic scaffold that can be extended with wallet connectivity,
transaction simulations, and other blockchain-related UI features.

---

## Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm (v9+)

### Setup

```bash
# install dependencies
npm install

# start development server
npm run dev
```

The app will be available at `http://localhost:3000`.

### Available Scripts

* `npm run dev` – start development mode with hot reloading
* `npm run build` – compile production build
* `npm run start` – run production build locally
* `npm run lint` – run Next.js/ESLint checks
